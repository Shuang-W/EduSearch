# EduSearch Frontend

### Installation Guide
```sh
npm install -g @angular/cli
npm install
```

### Database Connection
Database details are set in files **src/environments/\*.ts**

### Lint Code
```sh
ng lint
```

### Run Development Build
```sh
ng serve
```

### Run Production Build
```sh
ng build --prod --aot --build-optimizer --named-chunks --output-hashing all
```
